解析script标签 主要是用于在html页面中插入js的方法 它主要有以下属性 charset 属性 少用 defer 属性 延迟javascript 执行 先加载html页面再执行 language 属性 开发语言的版本 少用 src 属性 多用 连接外部的javascript文件 type 属性

在script标签中 使用alert 弹出 script

script标签中 使用src 属性 那么 原来的标签内的代码 就不会执行的

当浏览器禁用javascript时 怎么处理 使用 noscript标签
